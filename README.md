# Dino Game

![dino](./img/dino-game.png)

## 👋 Welcome

Come and Play this interactive fully responsive dino game. Jump over cacti and achieve the highest score.

## 🔋 Features

👉 **Play using keyboard**

👉 **dynamic scoring system**

👉 **background music and sound effects**

👉 **restart game**
